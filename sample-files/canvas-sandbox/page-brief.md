# Canvas Sandbox Page Brief

Create a Canvas page for a sandbox course.

## Page Purpose

Introduce a short activity where participants compare three ways of using AI in
course preparation:

- Asking for advice.
- Drafting course material.
- Building a small interactive activity.

## Required Page Sections

- Short introduction.
- Activity instructions.
- Reflection questions.
- Accessibility reminder.

## Tone

Clear, practical, and calm. Avoid hype.
