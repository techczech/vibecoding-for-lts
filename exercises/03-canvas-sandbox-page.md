# Exercise 3: Draft A Canvas Sandbox Page

## Goal

Explore how Codex could help prepare a Canvas page without touching live course
data.

## Safety Boundary

Use a sandbox course. Store credentials outside the repository. Do not use live
student data.

## Task

1. Read `sample-files/canvas-sandbox/page-brief.md`.
2. Ask Codex to draft a Canvas page body from the brief.
3. Ask Codex to identify the API endpoint it would need.
4. Stop before publishing unless the sandbox and token are ready.

## Reflection

- What needs human approval before publishing?
- What permissions should the API token have?
- What rollback plan would you want?
