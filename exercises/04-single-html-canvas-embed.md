# Exercise 4: Package A Web Activity As Single HTML

## Goal

Understand the difference between a normal web app and a single-file HTML
activity that can be uploaded to Canvas.

## Task

1. Choose a small web activity.
2. Ask Codex to identify its runtime dependencies.
3. Ask Codex to produce a single-file HTML version.
4. Open the file from disk.
5. Upload it to a Canvas sandbox and embed it in a page.

## Reflection

- What broke when the app became one file?
- What worked better than expected?
- Who would maintain the file after course launch?
