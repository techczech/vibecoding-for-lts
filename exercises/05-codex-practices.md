# Exercise 5: Set Up Repeatable Codex Practice

## Goal

Turn one useful interaction into a repeatable project habit.

## Task

1. Create or review an `AGENTS.md` file for a project folder.
2. Add one clear first move.
3. Add one verification command or checklist.
4. Add one privacy or safety constraint.
5. Ask Codex to follow those instructions on a small task.

## Reflection

- Which instruction made the biggest difference?
- What should become a reusable skill?
- What should stay project-specific?
