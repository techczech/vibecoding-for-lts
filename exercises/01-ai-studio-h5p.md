# Exercise 1: Recreate An H5P-Style Activity

## Goal

Use Google AI Studio to prototype a small learning object from a familiar
interaction pattern.

## Task

1. Choose a simple H5P-style activity: flash cards, branching scenario,
   drag-and-drop, timeline, or matching.
2. Ask AI Studio to create a single-page web activity.
3. Test the interaction as a learner.
4. Note what works, what is missing, and what would need human review.

## Reflection

- What learning decision did the AI need from you?
- What accessibility issue might be hidden in the prototype?
- What would make this safe enough to use in a course?
