# Exercise 2: Ask Codex To Inspect A Folder

## Goal

Use Codex to read a local folder and propose an organisation or visualisation.

## Task

1. Open this repository in Codex.
2. Use `sample-files/downloads-folder/` as a safe stand-in for a real Downloads
   folder.
3. Ask Codex to inspect the files and describe the patterns.
4. Ask for a proposed folder structure or simple visualisation.
5. Review the proposal before allowing file edits.

## Reflection

- What did Codex infer from names and contents?
- What should it have asked before changing anything?
- What would be different with your real files?
