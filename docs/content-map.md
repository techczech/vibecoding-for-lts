# Content Map

This map tracks the first usable version of the training material.

## Current Starting Set

- `README.md`: repository overview, build path, and publication checks.
- `content/source-outline.md`: public-safe source outline adapted from the
  private draft.
- `content/deck.yml`: canonical deck metadata and section order.
- `content/slides/*.md`: one editable slide source per slide.
- `exercises/*.md`: facilitator-ready exercise sheets.
- `sample-files/`: synthetic files for participant practice.
- `src/`: browser slide deck with participant notes.
